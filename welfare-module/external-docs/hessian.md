# 理解 Hessian

## Hessian 是什么

> 🚩 [百度百科](https://baike.baidu.com/item/Hessian/2385196?fr=aladdin)：Hessian 是一个轻量级的 remoting onhttp 工具，使用简单的方法提供了 RMI 的功能。 相比 WebService，Hessian 更简单、快捷。采用的是二进制 RPC 协议，因为采用的是二进制协议，所以它很适合于发送二进制数据
